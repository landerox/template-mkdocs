# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this repository, please report it by emailing **[security@example.com](mailto:security@example.com)**.

Please do not open public issues for security vulnerabilities.

## Scope

This repository contains documentation only. Security concerns might include:

- Exposed credentials or secrets in documentation examples
- Malicious links or references
- Vulnerabilities in dependencies (MkDocs plugins)

## Response

We will acknowledge receipt within 72 hours and provide updates as the issue is investigated. We aim to resolve security issues promptly and will work with you to ensure responsible disclosure.
